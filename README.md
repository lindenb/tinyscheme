## Extended functions

`(rand) => 0.89789` create a random float bewteen 0.0 and 1.0. 

`(tolower "ABC\n") => "abc\n" ` converts a string to lowercase

`(toupper "abc\n") => "ABC\n" ` converts a string to uppercase

`(trim " ABC  ") => "ABC" ` remove leading/trailing spaces
